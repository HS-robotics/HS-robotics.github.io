Project Overview

Introduction and Problem Definition

  • Addressed the challenge of safely operating therapeutic devices in isolated environments

Solution

  • Developed a GUI for enhanced user interaction and system visualization
  • Created a modular data structure to support multi-robot scalability

Results

  • Successfully integrated multiple therapeutic robots into a unified teleoperation system
  • Demonstrated high reliability in isolated intensive care scenarios
  • Achieved seamless real-time monitoring and operational control
